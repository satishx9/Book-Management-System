## Book-Management-System
Python CLI Book Management System: Manage books via a menu-driven interface with CRUD operations (Add, List, Search, Update, Delete) and price sorting. Uses modular functions, lists, and dictionaries to separate logic from UI. Ideal for practicing data handling and loops.
# Features
- Add new books
- List all books
- Search books by name
- Mark books as read
- Update book prices
- Delete books
- Sort books by price

# Technologies Used
- Python
- Lists and Dictionaries
- Modular Programming
- Command Line Interface (CLI)
